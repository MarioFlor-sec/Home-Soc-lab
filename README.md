# Home Soc lab
Building a Home SOC using Wazuh, Windows, and Linux to develop hands-on cybersecurity and security monitoring skills.

## Project Overview

This project documents my process of building a home Security Operations Center (SOC) using Wazuh, Windows, and Linux.

The goal of this project is to gain hands-on experience with:

- Security monitoring
- Log analysis
- SIEM tools
- Windows event logs
- Linux administration
- Threat detection fundamentals

## Lab Architecture

Planned architecture:

Windows VM → Wazuh Agent → Wazuh Server (Ubuntu VM) → Wazuh Dashboard

A visual network diagram will be added as the project progresses.
